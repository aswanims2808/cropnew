# 🌿 Plant Disease Detection AI

An AI-powered web application that identifies plant diseases from leaf images. Built using Python, TensorFlow, and Gradio, this tool provides instant diagnostic feedback and visual explanations to help monitor crop health.

## 🚀 Features
- **Instant Diagnosis:** Upload an image of a plant leaf to get an immediate prediction.
- **Visual Explainability:** Uses Grad-CAM to generate a heatmap, highlighting the areas of the leaf the AI focused on for its diagnosis.
- **User-Friendly Interface:** Built with Gradio for a clean, intuitive web experience.
- **Highly Accurate:** Trained on a comprehensive dataset of various crop species and disease classes.

## 🛠 Tech Stack
- **Frontend/UI:** Gradio
- **Machine Learning:** TensorFlow / Keras
- **Image Processing:** OpenCV, NumPy, Pillow

## 📋 Prerequisites
Ensure you have **Python 3.10+** installed. It is highly recommended to use a virtual environment.
